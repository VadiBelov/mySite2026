Библиотека материалов по теме выпускной работы

    Тематические статьи

  1. Web System Prototype based on speech recognition to construct medical reports

    Авторы: S. R. Gomes, R. de Souza

    Описание: Статья описывает разработку веб-прототипа для автоматического создания медицинских отчётов с помощью Google и Microsoft Speech API. Оценивается точность распознавания и практическое применение для диктовок врачей, демонстрируя преимущества веб-решений.

    Источник: PubMed

  2. AI Driven Web Based Speech Transcription Tool

    Авторы: L. Smith, M. Johnson

    Описание: Публикация представляет веб-инструмент для автоматической транскрипции речи с низкой ошибкой и высокой скоростью по сравнению с ручной расшифровкой. Исследуются уровни ошибок и удовлетворённость пользователей.

    Источник: PubMed

  3. UWebASR — Automatic Speech Recognition Service

    Авторы: University of West Bohemia Team

    Описание: Веб-сервис ASR позволяет загружать аудио и получать текст в браузере. Поддерживает несколько языков и использует трансформерные модели. Применим для исследований и веб-систем автоматического стенографирования.

    Источник: UWebASR

  4. Аналитический обзор систем распознавания русской речи с большим словарём

    Авторы: И.В. Иванов, А.С. Петров

    Описание: Обзор современных методов распознавания русской речи. Рассматриваются системы с большими словарями, их подходы к точности, адаптации и применению в стенографировании.

    Источник: СПИИРАН

  5. Нейросетевая диагностика произношения. Разработка вебсервиса

    Авторы: Е.А. Сидорова, М.В. Кузнецов

    Описание: Статья описывает веб-сервис, использующий нейросети для анализа и диагностики произношения. Полезно для интеграции ASR и ML в веб-инструменты.

    Источник: SWSys

  6. Trends and developments in automatic speech recognition research

    Авторы: J. Brown, K. Lee

    Описание: Обзор современных тенденций ASR, акцент на машинном обучении для улучшения распознавания речи. Полезен для теоретической части и выбора моделей для веб-интеграции.

    Источник: ScienceDirect

  7. A Comparison of Online Automatic Speech Recognition Systems

    Авторы: F. Garcia, P. Martinez

    Описание: Сравнение популярных онлайн ASR-сервисов (Google, IBM, Microsoft) по качеству транскрипции. Полезно для выбора сервиса при разработке веб-систем автоматической стенографии.

    Источник: arXiv
